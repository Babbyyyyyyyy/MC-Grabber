# MC-Grabber

---> Educational Purposes Only <---

C# Minecraft & Lunar Grabber (Session ID & Access Token & UUID & Username)
   * Discord Webhook Based.
      * Support Me By A Star

## Features 

* Supports All Clients.
* Grabs UUID.
* Grabs Username.
* Grabs Session ID.
* Grabs Access Token.
* Sends All Of Them To A Discord Webhook.
